## Description

Docker image of [Foreman](http://theforeman.org/)

## Setup

```sh
cd /opt
git clone https://github.com/solution-libre/docker-foreman.git foreman
cd foreman
```
## Usage

```sh
cd /opt/foreman
docker-compose up -d
```
